# 我的餐廳清單

## 簡介

可以瀏覽餐廳名稱、類別和評分，並可查看詳細資訊，以及可讓使用者透過關鍵字搜尋。

## 功能

1. 瀏覽所有餐廳的名稱、類別、相片和評分
2. 可點擊觀看更多資訊，如餐廳介紹、電話和地址
3. 可利用關鍵字搜尋
4. 可透過連結至該餐廳的 Google Map 網頁

## 開始使用

1. 進入"restaurant_List"資料夾
2. npm install
3. npm run dev
4. 打開瀏覽器 http://localhost:3000
5. 關閉伺服器請按 ctrl + c

## 使用版本

1. node.js: 14.15.5
2. express: 4.16.4
3. express-handlebars: 3.0.0
